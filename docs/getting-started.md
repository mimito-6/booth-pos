# 出攤前準備（給賣家，不用寫程式）

建議**在家先設定好**，現場零學習即用。

## 1. 開啟與安裝
- 用手機瀏覽器打開 Demo 連結（或你自架的網址）。
- 點瀏覽器選單「**加到主畫面**」→ 之後從主畫面開啟就像 App，且**會場沒網路也能用**。

## 2. 建立你的商品
1. 主頁 →「**庫存管理 (STOCK)**」。
2. 先建分類（右上 🏷️）：例如 貼紙 / 壓克力 / 吊飾。
3. 「＋ 新增商品」：填名稱、價格、初始庫存、選分類，可拍照 / 上傳圖片。
4. 有「N 張 M 元」優惠？在「批量優惠」填數量與整組價（例：4 張 100）。
5. 有套組 / 福袋？「＋ 新增套組」選內含商品與數量、設定整組價（會自動扣對應庫存）。

> 不想一個個建？把別人的設定包連結貼進來（見下方第 6 點），或在「設定」匯入 JSON。

## 3. 設定收款方式
主頁 →「**收款設定 (PAY)**」：保留現金，啟用 Line Pay / PayPay / 街口…，可上傳收款 QR 圖（結帳時放大給客人掃）。

## 4. 設定活動場次
主頁 →「**活動設定 (EVENT)**」：新增場次（例：CWT Day1 2026-08-XX），填**開攤備用零錢**（收攤對帳會用到）。多天活動就建多個場次，現場切換。

## 5. 現場結帳
主頁 →「**開攤一鍵啟動**」：點商品加入購物車 → 查看明細 → 結帳 → 選付款方式 → 現金輸入收到金額（或按面額快捷）→ 完成。
- 想標記送的特典 / 小禮：在明細點該行的「特典」。
- 好友價 / 湊整：點該行「改價」，或用「🧮 手動調整 / 湊整」。

## 6. 收攤
- 「**販售紀錄 (RECORD) → 收攤對帳**」：輸入實點現金，對照「現金應有」差額；看各付款方式小計與剩餘庫存。
- **務必到「設定 → 匯出全部資料」備份一次**（清快取 / 換手機會清空資料）。

## 分享你的攤位
「設定 → 產生分享連結」會把商品 / 主題打包成一條網址（不含交易與顧客資料），貼給朋友 60 秒複製整攤。
